# Giới Thiệu

Đây là dự án cá nhân của tôi, nơi lưu trữ các thử nghiệm, học tập và phát triển kỹ năng lập trình.

## Mục đích

- Thực hành và cải thiện kỹ năng coding
- Lưu trữ các dự án nhỏ và ý tưởng
- Tài liệu tham khảo cá nhân

## Nội dung

Repository này có thể bao gồm:
- Code mẫu và ví dụ
- Các bài tập thực hành
- Dự án thử nghiệm
- Ghi chú và tài liệu học tập
